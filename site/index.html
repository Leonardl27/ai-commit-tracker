Methodology

This dashboard detects AI-assisted commits by searching for Co-Authored-By git trailers containing known AI tool email addresses. Currently tracked: Claude (Anthropic), OpenAI Codex CLI, Gemini Code Assist (Google), and Cursor (Anysphere). Only repos with 50+ stars are included to filter noise.

Note: GitHub Copilot does not add Co-Authored-By trailers to commits, so it is not represented here. Some tools allow users to disable attribution, so actual usage may be higher than shown.

Data sourced from the GitHub Commit Search API. Updated daily.
